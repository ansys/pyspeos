Add Display Source to core layer

Display source is now available in the core.source module following the same architecture as existing sources (Luminaire, Surface, RayFile). The implementation includes:

- New `SourceDisplay` class with full support for all display source properties
- Default values matching SCDM Speos defaults
- Comprehensive unit tests with >85% code coverage
- Documentation example in examples/core/display-source.py

Features:
- Set image file (PNG, JPEG, BMP, TIFF, RGB formats)
- Configure source dimensions (horizontal and vertical)
- Set luminous flux (luminance in cd/mÂ²)
- Configure contrast ratio (optional, infinite if not set)
- Choose color space: sRGB, Adobe RGB, or custom RGB
- Set intensity distribution (Lambertian and other patterns)
- Position and orientation control via axis system
