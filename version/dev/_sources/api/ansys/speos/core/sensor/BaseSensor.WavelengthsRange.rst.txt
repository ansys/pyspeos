.. py:class:: ansys.speos.core.sensor.BaseSensor.WavelengthsRange(wavelengths_range, default_parameters = None, stable_ctr = False)

   
   Range of wavelengths.

   By default, a range from 400nm to 700nm is chosen, with a sampling of 13.

   :Parameters:

       **wavelengths_range** : :obj:`ansys.api.speos.sensor.v1.common_pb2.WavelengthsRange`
           Wavelengths range protobuf object to modify.

       **default_parameters** :         :obj:`ansys.speos.core.generic.parameters.WavelengthsRangeParameters`, :obj:`optional`
           If defined the values in the sensor instance will be overwritten by the values of the
           data class

       **stable_ctr** : :ref:`bool <python:bltin-boolean-values>`
           Variable to indicate if usage is inside class scope
