





:class:`LayerTypeSequence`
==========================


.. py:class:: ansys.speos.core.sensor.BaseSensor.LayerTypeSequence(layer_type_sequence, default_parameters = None, stable_ctr = False)

   
   Type of layer : Sequence.

   Includes in the result one layer per sequence.
   By default, the sequence is defined per geometries, with a maximum number of 10 sequences.

   :Parameters:

       **layer_type_sequence** :         :obj:`ansys.api.speos.scene.v2.scene_pb2.Scene.SensorInstance.LayerTypeSequence`
           LayerTypeSequence protobuf object to modify.

       **default_parameters** : :obj:`ansys.speos.core.generic.parameters.LayerBySequenceParameters`, :obj:`optional`
           If defined the values in the sensor instance will be overwritten by the values of the
           data class

       **stable_ctr** : :ref:`bool <python:bltin-boolean-values>`
           Variable to indicate if usage is inside class scope











   .. rubric:: Notes

   **Do not instantiate this class yourself**, use set_layer_type_sequence method available in
   sensor classes.
