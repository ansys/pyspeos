





:class:`LayerTypeFace`
======================


.. py:class:: ansys.speos.core.sensor.BaseSensor.LayerTypeFace(layer_type_face, default_parameters = None, stable_ctr = False)

   
   Type of layer : Face.

   Includes in the result one layer per surface selected.
   By default, a filtering mode by last impact is chosen.

   :Parameters:

       **layer_type_face** :         :obj:`ansys.api.speos.scene.v2.scene_pb2.Scene.SensorInstance.LayerTypeFace`
           LayerTypeFace protobuf object to modify.

       **default_parameters** : :obj:`ansys.speos.core.generic.parameters.LayerByFaceParameters`, :obj:`optional`
           If defined the values in the sensor instance will be overwritten by the values of the
           data class

       **stable_ctr** : :ref:`bool <python:bltin-boolean-values>`
           Variable to indicate if usage is inside class scope
