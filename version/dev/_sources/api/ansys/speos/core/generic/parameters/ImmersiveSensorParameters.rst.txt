





:class:`ImmersiveSensorParameters`
==================================


.. py:class:: ansys.speos.core.generic.parameters.ImmersiveSensorParameters

   
   Parameters for :class:`~ansys.speos.core.sensor.SensorImmersive`.


   :Parameters:

       **sampling** : :class:`python:int`, :obj:`optional`
           Horizontal and vertical number of pixels for a face.
           By default, ``600``.

       **integration_angle** : :class:`python:float`, :obj:`optional`
           Integration angle in degrees for direct simulations.
           By default, ``5.0``.

       **wavelengths_range** : :obj:`ansys.speos.core.generic.parameters.WavelengthsRangeParameters`, :obj:`optional`
           Spectral range to use for simulation.
           By default, a range from 400nm to 700nm with sampling of 13.

       **axis_system** : :class:`python:list`\[:class:`python:float`], :obj:`optional`
           Position of the sensor (Ox Oy Oz Xx Xy Xz Yx Yy Yz Zx Zy Zz).
           X corresponds to Front direction. Y corresponds to Top direction.
           By default, ``[0, 0, 0, 1, 0, 0, 0, 1, 0, 0, 0, 1]``.

       **layer_type** : :obj:`Union`\[:obj:`LayerTypes.none`, :obj:`LayerTypes.by_source`], :obj:`optional`
           Layer separation type.
           By default, ``LayerTypes.none``.

       **exclude_front** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude front face.
           By default, ``False``.

       **exclude_back** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude back face.
           By default, ``False``.

       **exclude_left** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude left face.
           By default, ``False``.

       **exclude_right** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude right face.
           By default, ``False``.

       **exclude_top** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude top face.
           By default, ``False``.

       **exclude_bottom** : :ref:`bool <python:bltin-boolean-values>`, :obj:`optional`
           Exclude bottom face.
           By default, ``False``.

       **interocular_distance** : :class:`python:float`, :obj:`optional`
           Distance between the left and right eyes, in millimeters.
           By default, None.
