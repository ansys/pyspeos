           Database to link to.

   .. rubric:: Examples

   >>> from ansys.speos.core.speos import Speos
   >>> from ansys.speos.core.kernel.intensity_template import (
   ...     ProtoIntensityTemplate,
   ... )

   >>> int_t_db = speos.client.intensity_templates()

   >>> int_t_message.cos.total_angle = 170
