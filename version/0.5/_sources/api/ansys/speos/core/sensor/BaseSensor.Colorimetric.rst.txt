.. py:class:: ansys.speos.core.sensor.BaseSensor.Colorimetric(sensor_type_colorimetric, default_values = True, stable_ctr = False)

   
   Type of sensor : Colorimetric.

   This kind of sensor will generate color results without any spectral data or layer
   separation in lx or W//m2.
   By default, it uses a default wavelengths range.

   :Parameters:

       **sensor_type_colorimetric** : :obj:`ansys.api.speos.sensor.v1.common_pb2.SensorTypeColorimetric`
