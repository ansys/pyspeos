





:class:`Gaussian`
=================

.. py:class:: ansys.speos.core.intensity.Intensity.Gaussian(gaussian, gaussian_props, default_values = True)

   
   Intensity of type: Gaussian.

   By default, full width at half maximum following x and y are set at 30 degrees, and total angle at 180 degrees.
   By default, no axis system is chosen, that means normal to surface map.

   :Parameters:

       **gaussian** : :obj:`ansys.api.speos.intensity.v1.IntensityTemplate.Gaussian`
           Gaussian to complete.

       **gaussian_props** : :obj:`ansys.api.speos.scene.v2.scene_pb2.Scene.SourceInstance.IntensityProperties.GaussianProperties`
           Gaussian properties to complete.

       **default_values** : :ref:`bool <python:bltin-boolean-values>`
           Uses default values when True.
