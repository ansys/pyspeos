.. py:class:: ansys.speos.core.sensor.BaseSensor.LayerTypeFace(layer_type_face, default_values = True, stable_ctr = False)

   
   Type of layer : Face.

   Includes in the result one layer per surface selected.
   By default, a filtering mode by last impact is chosen.
