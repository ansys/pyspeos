





:class:`LayerTypeSequence`

.. py:class:: ansys.speos.core.sensor.BaseSensor.LayerTypeSequence(layer_type_sequence, default_values = True, stable_ctr = False)

   
   Type of layer : Sequence.
   Includes in the result one layer per sequence.
   By default, the sequence is defined per geometries, with a maximum number of 10 sequences.


   :Parameters:

       **layer_type_sequence** : :obj:`ansys.api.speos.scene.v2.scene_pb2.Scene.SensorInstance.LayerTypeSequence`
